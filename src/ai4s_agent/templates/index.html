AI4S Agent 控制台

用于规划、数据审阅、风险确认、运行监控和报告审阅的渐进式界面。默认只展开当前步骤。

渐进式流程
当前页面是直接从模板文件打开的，API 请求不会工作。请通过 Flask 服务打开： PYTHONPATH=src .venv/bin/python -m flask --app ai4s_agent.app:create_app run --port 5005， 然后访问 http://127.0.0.1:5005/。
